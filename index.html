Workflows I've built

A few automations I've designed end-to-end — shown as the trigger → action chain that powers each one.

AI Restaurant Booking Automation

n8n + Gemini

I built an AI-powered restaurant booking automation using n8n and Google Gemini. Customers can book a table by sending their name, date, time, and number of guests in chat. The AI agent extracts the data, checks availability in Google Sheets, prevents double bookings, and confirms reservations automatically. The workflow includes: Chat Google Gemini model Memory buffer Google Sheets integration The system ensures bookings are only added if the time slot is available and never overwrites existing reservations.

Chat Message AI Agent Read/Add Booking Tool Booking Sheet
No staff needed to take bookings

Automated Inventory Monitoring & Low Stock Alert System

n8n + Gemini

I built an automated inventory monitoring system using n8n, Google Sheets, and Google Gemini. The workflow runs weekly, checks stock levels against predefined thresholds, and detects low-stock items automatically. When inventory falls below the limit, the system aggregates affected products and sends a structured alert via Gmail. This automation prevents stockouts, reduces manual checks, and improves purchasing efficiency.

Schedule Filter Low Stock AI Agent Gmail Alert
Zero missed reorders

AI Customer Feedback Classification & Routing System

n8n + Gemini

I built an AI-powered feedback automation system using n8n and Google Gemini. When a new row is added to Google Sheets, the workflow analyzes the customer’s feedback, classifies it by category (question, problem, suggestion) and area (kitchen, delivery, service), then updates the sheet automatically. Based on classification, the system routes notifications to the appropriate department via Gmail, enabling faster response and structured feedback management.

Sheets Row Added AI Categorization Route by Area Team Email
Feedback reaches the right team instantly

Automated Client Onboarding & CRM Workflow

n8n + HubSpot

I built a fully automated client onboarding system using n8n. When a form is submitted via Jotform, the workflow validates input data, creates a contact in HubSpot CRM, and sends structured team notifications via Telegram. The system delivers welcome emails, loyalty guides, timed follow-up offers, and status updates automatically. It also updates CRM stages and alerts the team when onboarding is complete, ensuring a smooth and professional customer journey.

Jotform Submit Validate + CRM Create Staged Follow-ups CRM Status Update
Multi-day sequence, zero manual follow-up

ERP Order Analytics & Reporting Automation

n8n + Discord

I built an analytics automation workflow using n8n that connects to a custom ERP system via HTTP request. The system filters processing orders, stores structured data in Airtable, calculates weekly order totals and revenue using JavaScript, and automatically sends performance summaries to Discord. This workflow provides real-time order tracking, structured data storage, and automated reporting for better operational visibility and decision-making.

Schedule ERP API Pull Condition + Transform Record / Discord Report
Always-on reporting, no manual pulls

AI Real Estate Lead & Property Matching System

n8n + Gemini

I built an AI-powered real estate system using n8n, Airtable, and Google Gemini. The workflow captures property inquiries via webhook, stores seller data in Airtable, and notifies agents automatically. For buyers, an AI agent analyzes property descriptions, searches Airtable records, and confirms availability using structured tool calls. The system also logs voice call summaries and costs, enabling intelligent lead tracking and automated property management.

Webhook Lead Validate + Log AI Agent + Airtable Search Matched Reply
Instant lead-to-listing matching
